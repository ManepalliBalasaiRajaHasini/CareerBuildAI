# 🚀 CareerBuild AI

CareerBuild AI is an AI-powered career guidance platform developed using **Python, Flask, HTML, CSS, JavaScript, and Google Gemini API**. It helps students explore career paths, prepare for interviews, and receive personalized AI-based career guidance.

---

## 📌 Features

- 🤖 AI Career Guidance
- 💼 AI Mock Interview
- 📚 Career Roadmaps
- 🎯 Personalized Career Suggestions
- 💻 Beginner-Friendly User Interface
- 📱 Responsive Design

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Google Gemini API

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YourUsername/CareerBuild-AI.git
```

Move into the project folder

```bash
cd CareerBuild-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
