# 🚀 CareerBuild AI

CareerBuild AI is an AI-powered career guidance platform developed using **Python, Flask, HTML, CSS, JavaScript, and Google Gemini API**. It helps students explore career paths, prepare for interviews, and receive personalized AI-based career guidance.

---

## 📌 Features

- 🤖 AI Career Guidance
- 💼 AI Mock Interview
- 📚 Career Roadmaps
- 🎯 Personalized Career Suggestions
- 💻 Beginner-Friendly User Interface
- 📱 Responsive Design

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Google Gemini API

## Note

The `app.py` file is not included in this repository because it contains API integration and confidential API credentials.

The project source code (HTML, CSS, JavaScript, and templates) is available for reference.

Backend functionality can be configured by adding your own Google Gemini API keys.
